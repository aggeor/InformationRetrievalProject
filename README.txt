Εκτέλεση μέσω του search_engine.exe

1.Κάνουμε extract τον φάκελο text files σε κάποιο directory(εστω στο desktop)
2.Βρίσκουμε το αρχείο search_engine.exe που βρίσκεται στο project/exe/dist
3.Κάνουμε extract το αρχείο search_engine.exe στον φάκελο text files(που βρίσκεται
    στο desktop)
4.Ανοίγουμε το αρχείο search_engine.exe

Εκτέλεση μέσω του search_engine.py με command prompt(Χρειάζεται Python 3.5)

1.Κάνουμε extract τον φάκελο text files σε κάποιο directory(εστω στο desktop)
2.Βρίσκουμε το αρχείο search_engine.py που βρίσκεται στο project/src
3.Κάνουμε extract το αρχείο search_engine.py στον φάκελο text files(που βρίσκεται
    στο desktop)
4.Ανοίγουμε command prompt
5.Πηγαίνουμε στο directory που βρίσκεται ο φάκελος text files
    ("cd desktop/text files")
6.Εκτελούμε το search_engine.py με την εντολή "python search_engine.py"
